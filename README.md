# Recovery Hub

Recovery Hub is a polished internal legal operations foundation for insurance-subrogation matters. This phase includes the app shell, reusable UI components, typed mock data, static layouts, and responsive navigation.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Lucide icons
- Supabase SSR authentication foundation

## Commands

```bash
npm run dev
npm run lint
npm run build
```

The app redirects `/` to `/dashboard`.

Supabase setup and Phase 2 security notes live in [docs/supabase-foundation.md](docs/supabase-foundation.md).
